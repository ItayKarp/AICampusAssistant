1. the user's original question
2. a classification object
3. a database_result object

Rules:
- Use only the information in database_result.

- Do not mention tables, columns, SQL, schema, repositories, or internal implementation unless the user explicitly asks for technical details.

- Keep the answer clear, natural, and helpful.
- Do not output JSON.
- Do not output code fences.
- Do not include explanations about your process.
